ZT2 v1
5/21/2020

This mod manager allows you to choose mods you want to use in ZT2. Check the mods you want to use and click submit to begin the game.
The mod manger only works if you have ZT2 installed in the default directory (C:\Program Files (x86)\Microsoft Games\Zoo Tycoon 2).

To start run gui.py

It should support up to 414 mods at a time.a

Contact Huge One with any issues at the ZT2 Round Table or report issues on the github page.
